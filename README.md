# BTI7054q
In diesem Modul werden die folgenden Themen und Inhalte behandelt:

* HTML (Formulare, Tabellen, Layout)
* Cascading Style Sheets (CSS)
* Protokoll HTTP (HyperText Transfer Protocole)
* Basissyntax von PHP (Typen, Bedingungen, Schleifen, Funktionen, Klassen)
* PHP als Sprache für das Web (Formulare, Cookies, Sessions, HTTP-Header)
* MySql Datenbanken (phpMyAdmin, Bibliotheken zur Einbindung von MySQL in PHP)
* Javascript
* Konfiguration des Apache HTTP Daemons
* Weiterführende Themen: Reguläre Ausdrücke, dynamische Bilder, PDF-Generierung, Web Services, Einführung in Ajax
